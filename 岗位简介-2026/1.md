# 岗位 JD 整理

## 一、全栈工程师（C#/.NET）
### 岗位职责
1.  负责前端模块开发与迭代，使用 Angular 框架实现页面交互、数据渲染及工程化优化，保障前端体验与稳定性。
2.  参与系统数据库设计，负责 SQL 性能调优，包括慢查询分析、索引优化、执行计划优化、数据库瓶颈排查等，提升数据处理效率。
3.  负责消息队列（Kafka/MQ）的集成、使用与维护，实现系统异步通信、流量削峰、服务解耦等核心场景开发。
4.  参与系统与 SAP 第三方系统的接口对接、集成开发及联调测试，保障数据互通与业务流程闭环。
5.  负责服务部署、容器化运维相关工作，配合团队完成 K8s 环境下的服务发布、扩容与问题排查。
6.  遵循开发规范，编写可维护、可扩展的代码，完成单元测试、接口文档编写及技术难点攻关。

### 任职要求
1.  本科及以上学历，计算机、软件工程等相关专业，**8 年及以上 C#/.NET 全栈开发经验**。
2.  精通 C#/.NET Core 2.2 及以上版本开发，熟练掌握 Web API、依赖注入、中间件等核心技术。
3.  精通 Angular 前端框架，熟悉 TypeScript、HTML5、CSS3，具备独立开发企业级前端项目能力。
4.  熟练掌握关系型数据库，具备扎实的 SQL 调优能力，能独立完成慢 SQL 分析、索引设计、数据库性能优化。
5.  具备 Kafka 或主流 MQ（RabbitMQ/RocketMQ）实际项目使用经验，了解消息生产、消费、重试、死信队列等机制。
6.  熟悉接口开发与第三方集成，了解 OAuth2 协议及常用对接方案。

## 三、AI Agent 全栈工程师
### 核心挑战
1.  LLM Agent 在高并发、低延迟、强一致性教育场景的工程化落地。
2.  单机万级长连接 + 毫秒级推送 + 消息零丢失。
3.  AI 教学链路中推理性能、上下文管理、检索效率的极限优化。
4.  多模态（图文/语音）在真实学习流程中的可控与稳定性。
5.  千万级教育数据的索引、缓存、实时更新与回溯核心。

### 工作内容
1.  用「工程+AI」提升工程效率与质量，在日常开发中持续探索、实践和优化：
    - AI 辅助开发方案设计
    - AI 辅助编码与重构
    - AI 辅助定位复杂问题
2.  构建教育场景 LLM Agent（诊断、讲解、追问、个性化引导）。
3.  与 ML Engineer 协作：
    - 推理链路设计与优化
    - 后训练（RL/风格对齐/个性化适配）的工程支撑
4.  设计并实现高性能后端架构：
    - 并发控制、消息系统、缓存体系
    - 高可用、可观测、可扩展的系统设计
5.  把抽象模型能力，变成可复用、可维护、可规模复制的工程系统。

### 任职需求
1.  学历：985/211 及以上院校，优先理工科背景。
2.  有 AI 产品相关经验。
3.  简历中有 AI 相关项目/实践经历。
4.  具备基本的问题拆解、需求分析、方案设计与推动落地能力。
5.  逻辑清晰，能讲明白每个动作背后的 why。
