# 腾讯 AI 应用开发一面 - CSIG（腾讯元宝）
面试题目整理

---

1.  手撕 LRU
2.  实习拷打（15 min）
3.  了解哪些记忆压缩机制
    - 如果工作流流程较长，多工具调用返回内容占据大量的上下文窗口，你会怎么管理上下文
    - 看你项目做了三层记忆，你讲讲短期记忆如何与用户进行关联，比如用户两次询问间隔了好几天如何维护记忆
    - 中期记忆要变成长记忆的标准是什么
    - 如果变成了长期记忆就一直呆在数据库中吗？
4.  讲讲 React 的缺陷
    - 说说这些缺陷应该如何优化
5.  实习中我看你基于业务场景进行了多智能体拆分，讲讲哪些场景适合多智能体拆分，如果拆分会带来哪些优势，又会造成哪些问题
6.  在做 RAG 的时候如果遇到文档排版混乱，比如双栏排布或者跨页表格这种问题怎么解决
7.  用户上传文档如何进行文档去重
    - 文档更新了，如何同步更新数据库
    - 做向量缓存如何保证缓存一致性？
    - 有没有分析过缓存命中率
    - 怎么优化 key 提高命中率
8.  有没有了解过 CLI 与 MCP 相比有什么不同
9.  有没有了解过 Open Claw 和 Claude Code 的底层
    - 你讲讲 Open Claw 的 Agent 架构是什么
    - 为什么它用 React 就可以，而 Claude Code 使用了比较复杂的架构
    - 讲讲 Open Claw 的记忆模式
    - 你看过 Claude Code 源码，你讲讲它是如何写 prompt 的
    - Claude Code 做了什么优化，保证了它在多轮对话下能做到任务不丢
