## 二、回溯 vs DP vs 贪心 —— 三大"求方案"的区分

这是最容易混淆的三个模式，核心区别在于**题目问的是什么**：

```
问"所有方案/所有路径/所有组合"  → 回溯（必须穷举）
问"最优值/最大/最小/方案数"    → 动态规划（最优子结构）
问"能不能到达/最少步数贪心可证" → 贪心（局部最优 = 全局最优）
```

### 2.1 回溯（Backtracking）识别三要素

**触发词：**
- "所有可能"、"所有组合"、"所有排列"、"所有子集"
- "返回所有满足条件的…"
- N 皇后、数独、括号生成

**核心模板（选—递归—撤销）：**

```js
function backtrack(路径, 选择列表) {
    if (满足结束条件) { result.push([...路径]); return; }
    for (选择 of 选择列表) {
        做选择;
        backtrack(路径, 选择列表);
        撤销选择;  // ← 精髓：恢复现场
    }
}
```

**仓库回溯题：** 46（全排列）、39（组合总和）、22（括号生成）、51（N皇后）、90（子集II）

**口诀：**
```
回溯三连：选、递、撤
for循环里做选择，递归结束要回撤
路径深了触底返，排列组合都在这
```

### 2.2 动态规划（DP）识别三要素

**触发词：**
- "最长"、"最大"、"最小"、"最少"
- "方案数"、"有多少种方法"
- 两个字符串/序列的匹配问题
- 问题可分解为重叠子问题

**DP 四大类型（仓库中出现）：**

| DP 子类型 | 状态定义 | 代表题 |
|-----------|----------|--------|
| **线性 DP（一维）** | `dp[i]` = 以 i 结尾的最优值 | 53（最大子数组）、300（LIS）、32（最长有效括号） |
| **双串 DP（二维）** | `dp[i][j]` = s前i和p前j的结果 | 10（正则匹配）、1143（LCS） |
| **背包 DP** | `dp[i][w]` = 前i个物品容量w的最优值 | 322（零钱兑换） |
| **树形 DP** | 后序遍历，子树返回状态给父 | 124（最大路径和）、543（直径） |
| **区间 DP** | `dp[i][j]` = 区间[i,j]的最优值 | 5（回文子串）、486（预测赢家）、877（石子游戏） |

**核心模板（三步走）：**

```js
// 1. 定义状态：dp[i] 代表什么？
// 2. 转移方程：dp[i] = f(dp[i-1], ...)
// 3. 初始条件：dp[0] = ?
for (let i = 1; i <= n; i++) {
    dp[i] = max(dp[i-1] + nums[i], nums[i]); // 以最大子数组为例
}
```

**口诀：**
```
DP 三步走：定义→方程→初值
一维线性看前驱，二维矩阵双循环
树形 DP 后序遍历，子节点先算再合并
区间 DP 对角线，长度递增往外推
```

### 2.3 贪心（Greedy）识别三要素

**触发词：**
- "最少步数"、"能否到达"
- 且局部最优可以证明是全局最优
- 常见：跳跃游戏、分发糖果、买股票

**贪心不可用时警示：**
- 零钱兑换（322）——贪心会错，必须 DP！
- 贪心需要**证明**，面试时如果拿不准，用 DP 兜底

**仓库贪心题：** 55（跳跃游戏）、45（跳跃游戏II）、121（买股票）、135（分发糖果）

**口诀：**
```
贪心看局部，一步一看路；
能到多远记最远，能省就省不回头；
拿不准的别硬贪，DP兜底最安全。
```

## 三、双指针家族（4 种变体）

### 3.1 对撞指针（首尾夹逼）

**场景：** 有序数组、两数之和变体、盛水容器、三数之和
**口诀：** `两边夹，谁小谁动；和大了右缩，和小了左扩`

```
题 11 盛水：低边决定容量，谁低缩谁
题 15 三数：固定一个，剩下对撞找两数之和
题 42 接雨水：左右 max 兜底，矮的那边结算
```

### 3.2 快慢指针（🐢🐇 龟兔赛跑）

**场景：** 链表判环、找中点、找倒数第 N
**核心：** 慢走 1，快走 2

```
慢一快二走到底，相遇必然有环在；
找中点时快先停，慢的位置就是中；
倒数第 N 先跑 N，快慢同走快到底。
```

**仓库题：** 141（环形链表）、142（入环点）、19（删除倒数N）、234（回文链表找中点）

### 3.3 滑动窗口（同向双指针）

**场景：** 子串、子数组、连续区间问题
**核心：** 右扩满足条件，左缩破坏条件

```
右指针扩张加元素，左指针收缩踢元素；
满足条件记答案，不满足了再右扩；
窗口大小灵活变，最长最短都搞定。
```

**仓库题：** 3（无重复字符最长子串）、239（滑动窗口最大值→升级为单调队列）

### 3.4 分离双指针（归并）

**场景：** 合并两个有序数组/链表
**核心：** 各指一个数组，比大小依次放

**仓库题：** 88（合并有序数组）、21（合并有序链表）、2（两数相加）

## 五、二叉树五句诀

```
遍历三兄弟，前中后序递归写，迭代栈来模拟
层序 BFS 队列跑，每层 size 控制好
BST 性质要记牢，左小右大区间校验（min, max）
LCA 后序往上找，左右都有的就是它
树形 DP 单边返，最大路径两边和（左贡献+右贡献+自己）
```

**遍历选择决策：**

```
需要逐层处理 → BFS（队列 + 层 size）
需要自底向上（子树结果汇总给父节点）→ 后序（左右根）
需要自上而下（带路径信息下传）→ 前序（根左右）
BST 有序处理 → 中序（左根右）
```

**树题模式映射：**

| 模式 | 题目 | 核心思路 |
|------|------|----------|
| 递归遍历 | 94, 144, 145 | 递归三行；迭代用栈模拟 |
| BFS 层序 | 102, 199, 111 | 队列+记录 level |
| BST 校验 | 98 | 传 [min, max] 区间递归 |
| LCA | 235, 236 | 后序，左右非空即答案 |
| 路径枚举 | 112, 113 | 回溯，到叶子判断 |
| 树形 DP | 124, 543 | 后序，返回单边贡献，全局取两边和 |
| 构造树 | 108, 105 | 二分中点做根，递归左右 |

## 八、图 / 岛屿套路（BFS/DFS 网格）

```
岛屿数量 DFS 沉，遇到 1 就计数，上下左右递归淹（变0）
最大面积 BFS 扩，queue 入队逐层扩
朋友圈即并查集，连通分量数一数
```

**网格 DFS 模板（淹没法）：**

```js
function dfs(grid, i, j) {
    if (i<0 || j<0 || i>=m || j>=n || grid[i][j]==='0') return;
    grid[i][j] = '0';  // 淹没！
    dfs(grid, i+1, j);
    dfs(grid, i-1, j);
    dfs(grid, i, j+1);
    dfs(grid, i, j-1);
}
```

**仓库题：** 200（岛屿数量）、695（岛屿最大面积）、547（朋友圈/省份数量）

## 九、边界思考速查表

| 边界类型 | 常见场景 | 处理方式 |
|----------|----------|----------|
| **空输入** | 任何题 | 开头判 `if(!nums || !nums.length) return 0/[]/null` |
| **链表头结点** | 链表删除/插入 | dummy 节点 → `return dummy.next` |
| **数组越界** | 二分、滑动窗口 | `while(l <= r)` 而不是 `l < r` |
| **整数溢出** | 反转整数（7）、回文数（9） | JS 安全整数 `Number.MAX_SAFE_INTEGER`；其他语言用 long |
| **负数/0** | DP 初始值 | 初始化为 `-Infinity` 或特判 |
| **单节点/两节点** | 链表、树 | 单独处理：`if(!head || !head.next) return ...` |
| **栈空弹出** | 括号匹配、单调栈 | `pop()` 前判断 `stack.length` |
| **重复元素** | 三数之和、子集 | 排序后跳过：`if(i>0 && nums[i]===nums[i-1]) continue` |
| **环形链表** | 快慢指针 | 判 `fast && fast.next` 而非只判 `fast` |
| **递归爆栈** | 树/Tail recursion | 考虑迭代写法，或尾递归优化 |
| **grid 边界** | DFS/BFS 网格 | 先判越界再访问：`i>=0 && i<m && j>=0 && j<n` |

**边界总口诀：**
```
空值开头判，链表加 dummy；
数组防越界，递归防爆栈；
二分查等号，环链判两跳（fast && fast.next）；
负数初始慎，去重排序跳。
```

## 十、复杂度速判

| 复杂度 | 暗示 | 典型算法 |
|--------|------|----------|
| O(1) | 数学公式/直接计算 | 哈希查找 |
| O(logN) | "有序"、"搜索"、数据减半 | 二分搜索 |
| O(N) | 遍历一次 | 双指针、单调栈、贪心 |
| O(NlogN) | 排序、堆操作、归并 | 排序、堆排前K、链表归并 |
| O(N²) | 双循环、二维 DP | LCS、回文子串 DP |
| O(2^N) | 穷举所有组合 | 回溯（子集、排列） |
| O(N!) | 全排列级穷举 | N皇后 |
