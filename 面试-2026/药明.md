## 二、现场手撕面试题（一二面）
### 1. 二叉树 BFS/DFS
- LC102：二叉树层序遍历 → 实验批次分层输出 **Medium**
- LC236：二叉树最近公共祖先 → 实验层级树查询公共父节点 **Hard**

### 2. 单调栈
- LC739：每日温度 → 培养箱温度，求升温间隔天数 **Medium**
- LC84：柱状图中最大矩形 → 培养皿容积计算（常口述思路）**Hard**

### 3. DP
- LC70：爬楼梯 → 试剂分装方案计数 **Easy**
- LC1143：最长公共子序列LCS → **生信必考**，两条基因同源最长片段 **Medium**

### 4. 链表&设计（后端岗）
- LC141：环形链表 → 实验编号闭环校验 **Easy**
- LC146：LRU缓存 → 实验数据缓存淘汰（社招必手写）**Medium**

### 5. 字典树
- LC208：实现Trie前缀树 → 基因序列字典匹配 **Medium**

   - 生信开发：1道原题 + 1道基因改编算法题
